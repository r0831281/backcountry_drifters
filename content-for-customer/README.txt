CONTENT FILES FOR BACKCOUNTRY DRIFTERS WEBSITE
================================================================================

PURPOSE
-------
This directory contains text files with all the content from your website. 
These files make it easy for you to review and adjust the phrasing and content 
where needed.


WHAT'S INCLUDED
---------------
1. home-page.txt
   - All text content from the homepage
   - Includes hero section, stats, features, trip descriptions, and CTAs

2. about-page.txt
   - All text content from the About page
   - Includes guide bio, trip specialties, values, waters, and certifications

3. bookings-page.txt
   - Content from the Bookings page
   - Note: Individual trip details are managed in the admin dashboard

4. layout.txt
   - Header navigation content
   - Footer content including quick links

5. contact.txt
   - All contact information displayed on the site
   - Email, phone, location details


HOW TO USE THESE FILES
-----------------------
1. Review each file and identify any changes you'd like to make
2. Edit the text directly in these files
3. Mark your changes clearly (you can use CAPS, [CHANGE THIS], or highlights)
4. Send the edited files back with your requested changes
5. The developer will implement your content updates in the website


WHAT CAN YOU CHANGE?
---------------------
Feel free to adjust:
- Phrasing and wording
- Service descriptions
- Contact information
- Location details
- Certifications and credentials
- Any text that appears on the website

Please note:
- Layout and design changes require additional development work
- New pages or sections require additional development work
- Trip-specific content (individual trips) is managed through the admin dashboard


IMPORTANT NOTES
---------------
- These files represent static content that is coded into the website
- Dynamic content (like individual trip listings and testimonials) is managed 
  through the admin dashboard and is not included in these files
- Keep the section structure intact so we can easily map your changes back to 
  the website


NEED HELP?
----------
If you have questions about any of the content or want to discuss changes, 
please reach out to your developer.
